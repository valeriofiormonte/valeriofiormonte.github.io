UX Redesign – Acquisition Page

UX redesign of a high-traffic acquisition page with the goal of improving clarity and increasing user engagement.

Project Overview

Context

The page was generating traffic but showed a low start rate, indicating friction in the user journey.

Goal

Improve usability and clarity to increase form starts.

My Role

UX design, layout redesign, CMS implementation and performance analysis.

Process

I started with a qualitative analysis of user behavior and competitors, identifying key friction points. Based on these insights, I redesigned the layout focusing on hierarchy, readability and call-to-action visibility.

Outcome & Results

The redesign resulted in a +4pp increase in start rate over a two-month compared period before and after the change.
